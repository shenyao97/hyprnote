# char

Install the Char CLI from npm.

## Install

```bash
npm install -g char
```

For nightly builds:

```bash
npm install -g char@nightly
```

## Platforms

The npm package currently installs prebuilt binaries for:

- macOS Apple Silicon
- macOS Intel

## Usage

```bash
char --help
```
